# rhagoletis
command line decryption utility for iOS

```
Mach-O Decryptor
Usage: rhagoletis [OPERATION] [OPTIONS] [MACH-O]
Operation:
  -d, --decrypt           decrypt all encrypted pages
  -l, --list              list all the encrypted pages
Options:
  -o, --output            specify file output (otherwise in-place)
  -v, --verbose           be verbose
  -vv                     be even more verbose
  -h, --help              display this message
  ```
